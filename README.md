# AutoSEO

  Automated SEO content generation platform. Users enter a website URL, the system analyzes the business, generates keywords, creates a 30-day content plan, and writes expert SEO articles targeting Google and AI search engines. Includes WordPress publishing with AI-generated featured images.

  ## Stack
  - React + Vite (frontend), Express 5 (API), PostgreSQL + Drizzle ORM
  - OpenAI gpt-5 for content generation, gpt-image-1 for featured images
  - pnpm monorepo, TypeScript 5.9, Tailwind CSS v4, shadcn/ui
